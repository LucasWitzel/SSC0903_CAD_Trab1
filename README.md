# SSC0903_CAD_Trab1
Projeto do primeiro trabalho da disciplina de Computação de Alto Desempenho da USP São Carlos (1º semestre de 2026).

## Execução
Para compilar ambos os códigos: make all

Para executar o código sequencial: make runseq input=entrada.txt

Para executar o código paralelo: make runpar input=entrada.txt

Para remover os arquivos executáveis e objetos: make clean
